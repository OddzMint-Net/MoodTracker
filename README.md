# MoodTracker

MoodTracker is a Jetpack Compose Android app that helps users log their mood, reflect on their emotions, and receive AI-powered journaling prompts.

## Features

- Log moods with emoji-based mood selection
- Save mood history locally using Room
- AI-powered journaling prompts using Gemini
- Offline fallback support messages
- Clean Jetpack Compose UI
- MVVM architecture

## Tech Stack

- Kotlin
- Jetpack Compose
- Room
- ViewModel
- StateFlow
- Gemini API
- Gradle Kotlin DSL

## Privacy

Mood entries are stored locally on the device. The app does not use accounts, analytics, or cloud storage.
